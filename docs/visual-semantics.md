# Visual Semantics

## Colors

- **Blue** (#E6F2FF) = T (Technical)
- **Purple** (#F1E6FF) = A (Application)
- **Gray** (#F5F5F5) = 0 (Core)

## Line Styles

- **Solid** = Dependency (uses)
- **Dashed** = Data flow (passes data)
